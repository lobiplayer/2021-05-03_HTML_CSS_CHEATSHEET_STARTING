element

usage

h1,h2,h3,h4, h5, h6

headers, where h1 is the most important one and h6 te least important

paragraph for text

a

anchor. makes a hyperlink from text, photo or element. href='' describes the page where the user will be directed to.

img

img is a selfclosing tag. the src='' describes the path to the source of the picture. This can be a local file or an online file.

ol, ul

ordered list and unordered list. content is added with li elements.
